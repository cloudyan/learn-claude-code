## 我们要写什么

我们从 **v1** 开始（最实用的版本），然后逐步升级到 v2、v3、v4，最后用 **v0** 来收尾（最极简的版本）。

| 版本 | 主题 | 核心洞察 |
|------|------|----------|
| v1 | 模型即代理 | 模型是 80%，代码只是循环 |
| v2 | 结构化规划 | Todo 让计划可见 |
| v3 | 分而治之 | 子代理隔离上下文 |
| v4 | 领域专家 | Skills 注入专业知识 |
| v0 | 终极简化 | Bash 就是一切 |

## 先剧透一点：核心模式

所有 Agent，不管多复杂，本质都是这个循环：

```python
while True:
    response = model(messages, tools)
    if response.stop_reason != "tool_use":
        break
    results = execute(response.tool_calls)
    messages.append(results)
```

就这么简单。

其他一切——待办、子代理、权限、Skills——都是围绕这个循环的精化。
