# 《手写 Claude Code》小册子写作计划 - 完整版

## 核心定位
**真正从 0 到 1，学完能手写 Agent，也能入门 LangChain 1.x**

| 维度 | 设计 |
|------|------|
| **定价** | 9.9元 |
| **篇幅** | 12篇文章，每篇3000-5000字，共约5万字 |
| **形式** | 公众号排版 + 代码仓库配套 |
| **核心承诺** | "学完第12篇，你能手写 v4，也能写 LangChain 1.x" |
| **双语言** | Python 手写版 + LangChain 1.x TypeScript 对照版 |

---

## 写作原则

1. **手写优先**：先手写 Python 版本，理解核心逻辑
2. **源码对照**：然后对照 LangChain 1.x 源码，看工业级实现
3. **可运行**：每篇都有可直接运行的完整代码
4. **有思考**：不仅教"怎么做"，更教"为什么这么做"
