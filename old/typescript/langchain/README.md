# learn-claude-code-langchain

LangChain (TypeScript) implementations of the Claude Code learning series.

## Setup

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Configure environment variables:
   ```bash
   cp .env.example .env
   # Edit .env with your API keys
   ```

## Running Agents

- **v0: Bash is All You Need**
  ```bash
  npm run v0
  ```
- **v1: Model as Agent**
  ```bash
  npm run v1
  ```
- **v2: Structured Planning (Todos)**
  ```bash
  npm run v2
  ```
- **v3: Subagent Mechanism**
  ```bash
  npm run v3
  ```
- **v4: Skills Mechanism**
  ```bash
  npm run v4
  ```
